# 야돈 트레이닝 센터 (Slowpoke Training Center)

느긋한 야돈을 클릭하고 돌보며 야도란, 야돈킹까지 성장시키는 귀여운 포켓몬 트레이닝 인터랙티브 웹사이트입니다.

## 📂 폴더 구조
```txt
slowpoke-training-site/
├── index.html          # 메인 시맨틱 HTML5 구조
├── README.md           # 프로젝트 가이드
├── css/
│   └── style.css       # 파스텔톤 테마 디자인 시스템 및 CSS Keyframe 모션 가이드
└── js/
    └── script.js       # 실시간 상태(State) 관리, 파티클 이펙트, 진화 로직
```

## 🎮 핵심 기능
1. **인터랙티브 캐릭터 클릭**:
   - 야돈을 클릭할 때마다 성장 포인트(+10) 및 클릭 횟수가 증가합니다.
   - 클릭 좌표를 실시간 추적하여 귀여운 이모지(하트, 별, 사탕 등) 파티클이 사방으로 퍼지는 시각 효과를 제공합니다.
   - 클릭할 때마다 야돈이 통통 튀는 애니메이션(`bounce`)과 함께 실시간으로 상태 메시지가 변경됩니다.
2. **트레이닝 액션**:
   - **물가 산책**: 성장 포인트 +10 획득 및 춤추는 좌우 흔들림(`wiggle`) 모션 발동.
   - **맛있는 먹이 주기**: 성장 포인트 +15 획득.
   - **낮잠 자기**: 성장 포인트 +5 획득.
3. **진화 시스템**:
   - **1단계 야돈**: 0 ~ 99 pt
   - **2단계 야도란**: 100 ~ 199 pt (진화 시 전체 화면 축하 모달 및 반짝임 효과)
   - **3단계 야돈킹**: 200 pt 이상 (최종 진화 시 영리한 야돈킹으로 모습 변환)
4. **일일 미션 및 팁**:
   - 클릭 수, 먹이 주기 횟수, 50점 도달 여부를 실시간으로 감지하여 체크리스트의 완수 여부를 실시간으로 표현합니다.
   - 트레이닝 상태를 0점 상태로 리셋하고 초기화할 수 있는 관리 기능을 제공합니다.

## 🛠️ 기술적 특징
- **웹 표준 & 접근성**: HTML5 시맨틱 요소를 적극 채용하고 키보드 탭(Focus) 및 엔터/스페이스 키 조작에 완벽히 대응합니다.
- **순수 기술**: 외부 라이브러리(Tailwind, CDN 등)가 배제된 순수 HTML5/CSS3/Vanilla JS 구성으로 높은 성능과 확장성을 제공합니다.
- **반응형 웹**: 미디어 쿼리를 기반으로 데스크톱 2열 뷰포트에서 모바일 1열 뷰포트까지 유연하게 레이아웃이 최적화됩니다.
